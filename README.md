Fack-passwd is a simple tool to create a fake /etc/passwd file for testing and educational purposes. This
    tool generates a fake passwd file with random user entries, which can be useful for simulating user management scenarios without affecting real system files.
## Features
- Generates a fake /etc/passwd file with random user entries.
- Customizable number of user entries.
- Easy to use command-line interface.

   git clone
